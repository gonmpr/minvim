
## Custom configuration summary

### Leader keys

- `<leader>` = Space

### Insert mode

- `jj` → exit insert mode

### Navigation

- `<Tab>` → next buffer
- `<S-Tab>` → save (if modified) and close current buffer
- `<C-d>` → half-page down, centered
- `<C-u>` → half-page up, centered

### Ephemeral terminal

- `<leader>t` → open terminal buffer
- `<S-Tab>` (terminal mode) → exit terminal and return to previous buffer

### File explorer

- `<leader>e` → open/close netrw in left split (20%)

### Code navigation

- `<leader>g` → go to definition (custom implementation)

### Editing behavior

- auto-close quotes and brackets in insert mode
- smart indent and auto indent enabled
- visual mode line move:
  - `J` → move line down
  - `K` → move line up
- visual mode indent keeps selection:
  - `<` → indent left
  - `>` → indent right
